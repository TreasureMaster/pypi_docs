# Паттерны Flask

Определенные функции и взаимодействия достаточно распространены, поэтому вы найдете их в большинстве веб-приложений. Например, многие приложения используют реляционную базу данных и аутентификацию пользователей. Они откроют соединение с базой данных в начале запроса и получат информацию для вошедшего в систему пользователя. В конце запроса соединение с базой данных закрывается.

Эти типы шаблонов могут немного выходить за рамки самого **Flask**, но **Flask** упрощает их реализацию. Некоторые общие шаблоны собраны на следующих страницах.
